Greatest F1 Drivers Analysis

After I started watching Formula 1, I kept seeing debates on who is the greatest of all time is and I wanted to come up with my own conclusion. Formula 1 had so many eras with different regulations, points systems, and championship systems. I wanted to clear up the variability of the various eras and remove factors that I believe aren't necessarily a good indicator of skill, such as the fastest lap point. I used SQL to process the data and Tableau to create the visualizations. (Data Source) (Tableau Link)

Online Tutoring Application

While working for Elements of Knowledge, a small group of developers and I created an online educational application using React.JS, Express.JS, Node.JS and MySQL. Over 200 math programs were created to test, evaluate and improve elementary school students' math skills. We used Passport.JS and Oauth2.0 for user authentication and created a backend API to organize users' tests history. This allows students and teachers to easily evaluate their progress of each math skill.

Reaction Speed Test

Using Python, I created a game that tests users reaction speed. In a 3x3 grid, users have to click the highlighted squares as fast as possible. After a completed run, the user can input their desired name, their record will be recorded in the leaderboard. Tkinter was used for the GUI and pandas was used to record scores into the leaderboard.
